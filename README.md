## Getting Started

1. Run `npm install`
2. Run `npm run dev`

## Backend API

Run the API server in a separate terminal:

```bash
npm run api
```

The backend listens on `http://127.0.0.1:4000/api`. During Vite development,
frontend requests to `/api` are proxied to that server.

Available endpoints:

- `GET /api/health`
- `POST /api/auth/login`
- `GET /api/roles`
- `POST /api/roles`
- `GET /api/modules`
- `GET /api/users`
- `POST /api/users`
- `PATCH /api/users/:id`
- `GET /api/counties`
- `GET /api/facilities`
- `GET /api/device-types`
- `GET /api/dashboard/summary`
- `GET /api/inventory`
- `POST /api/inventory`
- `PATCH /api/inventory/:id`
- `GET /api/requisitions`
- `POST /api/requisitions`
- `GET /api/requisitions/:id`
- `PATCH /api/requisitions/:id`

Bootstrap super admin credentials:

- Username: `admin`
- Password: `admin123`

## Database Schema

The Node.js backend database schema is in `server/database`.

- `server/database/schema.sql` creates the PostgreSQL tables and indexes.
- `server/database/seed.sql` inserts development seed data.
- `server/database/tables.mjs` exports table names and status constants for Node.js code.
